====OVERVIEW====
The CASYOW Calculator App - Aldana Edition is a simple, responsive, and easy-to-use calculator designed for basic arithmetic calculations. 
Inspired by the classic CASIO calculator style, this web app allows users to perform addition, subtraction, multiplication, 
and division with a clean and intuitive interface.



====FEATURES====
*Basic Arithmetic Operations: Supports addition (+), subtraction (-), multiplication (*), and division (/).
*Equal Button: Calculate the result of the current expression with the = button.
*Clear Button: Clear the current input or reset the calculator with the C button.
*Responsive Design: Works seamlessly across different devices such as desktops, tablets, and mobile phones.
*Sleek Design: Inspired by the classic CASYOW calculator, with a custom interface designed by "Aldana Edition."
